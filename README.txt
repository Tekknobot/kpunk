kmusic (UI Toolkit) — Portrait Plugin UI Starter (Unity 2022.3.62f3)

Open this folder as a Unity project.

Scene:
- Assets/Scenes/Main.unity

What you get:
- UI Toolkit portrait layout matching the dark mockup direction
- ParameterBus + Parameter system (single source of truth)
- Reusable controls: KnobElement, FaderElement, StepGrid
- Preset Browser overlay scaffold
- Theme tokens via USS variables (easy to reskin)

Quick test:
1) Open Main scene
2) Press Play
3) Drag knobs/faders; watch values update
4) Toggle Preset Browser with the top preset bar button
